# 🪨📄✂️ Jokenpô em Python

Projeto simples em Python que simula o jogo **Pedra, Papel e Tesoura** contra o computador.

## 🎯 Objetivo

Praticar lógica de programação, controle de fluxo e tratamento de erros.

## ⚙️ Funcionalidades

* Escolha do usuário com validação
* Escolha aleatória do computador
* Sistema automático de vitória, derrota ou empate
* Tratamento de entradas inválidas

## 🧠 Conceitos aplicados

* Estruturas condicionais (`if`, `elif`, `else`)
* Laços de repetição (`while`)
* Tratamento de erros (`try/except`)
* Listas e dicionários
* Módulo `random`

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone <link-do-repositorio>
```

2. Acesse a pasta:

```bash
cd jokenpo
```

3. Execute o programa:

```bash
python jokenpo.py
```

Projeto desenvolvido como exercício prático para evolução em Python, com foco em lógica e boas práticas iniciais.
